# CSS Selectors Task

## Project Description

This project demonstrates the use of basic CSS selectors to style HTML elements. It includes headings, paragraphs, links, and lists styled using different CSS selectors such as element, class, ID, descendant, pseudo-class, and `nth-child` selectors.

## Files Included

* `index.html` – Contains the HTML structure.
* `style.css` – Contains all the CSS styling.
* `README.md` – Project documentation.

## Features

* Applied a background color to the webpage.
* Styled multiple headings using a class selector.
* Styled a specific heading using an ID selector.
* Styled paragraphs using element and ID selectors.
* Styled a specific word using a descendant selector.
* Styled specific list items using the `nth-child()` selector.

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. The webpage will display the styled output using CSS selectors.
